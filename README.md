# A2A-SIN-Google-Chat

> A2A agent for Google-Chat integration within the OpenSIN ecosystem.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Stars](https://img.shields.io/github/stars/OpenSIN-AI/A2A-SIN-Google-Chat)](https://github.com/OpenSIN-AI/A2A-SIN-Google-Chat)

## Overview

This repository provides an Agent-to-Agent (A2A) integration for Google-Chat, enabling seamless communication and automation within the OpenSIN-AI ecosystem.

## Quick Start

```bash
git clone https://github.com/OpenSIN-AI/A2A-SIN-Google-Chat.git
cd A2A-SIN-Google-Chat
npm install
npm start
```

## Features

- A2A protocol support for Google-Chat
- Automated message routing
- Integration with OpenSIN MCP servers
- Full observability via OpenSIN-Ledger

## Architecture

This agent connects to the OpenSIN orchestrator and handles Google-Chat-specific messaging through the A2A protocol.
